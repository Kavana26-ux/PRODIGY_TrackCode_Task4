# Social Media Sentiment Analysis
This project analyzes social media data to identify public sentiment and opinions towards topics or brands using Python and NLP techniques.

## Features
- Text preprocessing and cleaning
- Sentiment classification
- Positive, Negative, and Neutral analysis
- Data visualization using graphs

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

## Output
- Sentiment prediction
- Sentiment distribution chart
- Public opinion insights
